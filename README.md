# 🌸 CipherBloom

A secure real-time chat application with:

- AES Encryption
- Diffie-Hellman Key Exchange
- RSA Algorithm
- JWT Authentication
- bcrypt Password Hashing
- Socket.io Real-time Messaging

## Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
